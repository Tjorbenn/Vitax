# Vitax Thesis & Defense

This directory contains data from the master thesis related to this project.

## Directory Structure

- [data/](data/): Raw and processed data used for evaluation, including:
  - [user-evaluation/](data/user-evaluation/): Results from the user study.
  - [lighthouse/](data/lighthouse/): Performance benchmarks (Lighthouse reports) for Vitax and comparable software.
  - [ncbi-taxonomy/](data/ncbi-taxonomy/) & [genbank/](data/genbank/): Statistical data and analysis.
